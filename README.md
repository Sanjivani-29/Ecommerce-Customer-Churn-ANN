# E-Commerce Customer Churn Prediction

## Overview
This project predicts whether an e-commerce customer will churn using an Artificial Neural Network (ANN).

## Technologies Used
- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Scikit-learn

## Dataset
E-Commerce Customer Churn Dataset from Kaggle.

## Model
- Artificial Neural Network (ANN)
- Adam Optimizer
- Binary Crossentropy Loss
- Early Stopping

## Results
The model predicts customer churn and is evaluated using:
- Accuracy
- Confusion Matrix
- Classification Report
