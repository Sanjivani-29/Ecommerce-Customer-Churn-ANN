

# 🛒 E-Commerce Customer Churn Prediction 

## 📌 Project Overview

Customer churn prediction helps businesses identify customers who are likely to stop using their services. This project builds an Artificial Neural Network (ANN) using TensorFlow/Keras to predict customer churn based on customer information.

---

## 🎯 Objectives

- Predict customer churn accurately.
- Preprocess categorical and numerical data.
- Train an Artificial Neural Network (ANN).
- Evaluate model performance using multiple metrics.

---

## 📂 Dataset

**Dataset:** E-Commerce Customer Churn Dataset

Source: Kaggle

https://www.kaggle.com/datasets/samuelsemaya/e-commerce-customer-churn

## 📈 Model Architecture

- Input Layer
- Dense Layer (16 neurons, ReLU)
- Dense Layer (8 neurons, ReLU)
- Output Layer (1 neuron, Sigmoid)

Optimizer:
- Adam

Loss Function:
- Binary Crossentropy

Evaluation Metrics:
- Accuracy
- Confusion Matrix
- Classification Report
